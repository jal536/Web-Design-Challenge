Summary: Pyber Analysis

The purpose of this project was to analyze the ride sharing data from Pyber, a ride sharing business, given different location demographics, to see the differences in location type, number of and total fares, and number of divers per location type.


We imported data from two different csv files using pandas in a python environment, then subsequently plotted the data using matplotlib to visualize the data. We used both scatter plots to be able to see the distribution of data and pie graphs to look at the percentages of the data as a whole.
